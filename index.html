Get Started

Prerequisites

  • Install Leiningen

Get the Immutable Stack

Git checkout the immutable-stack so you have the demonstration app from David and Kovas (see Inspiration)

git clone https://github.com/KitchenTableCoders/immutable-stack.git

Install the Immute Template for Leiningen

This template lets you build a basic app - currently incomplete

Do the following to install the app: 

git clone https://github.com/immutable-hack/immute-template
cd immute-template
lein install
cd ..

Create a new app with the template:

lein new immute my-first-immute-app --snapshot

Inspiration

This project draws inspiration from:

  • David Nolen - who is an enthusiastic and friendly Clojure developer. He and his cohort, Kovas Boguta created The Immutable Stack.
  • Ruby On Rails - teaches us what to do (ease the transition to a new language and stack) and what not to do (railroad users into a brittle app framework which promotes low cohesion).
